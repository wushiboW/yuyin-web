## 🏔 关于屿音

**屿音** 是一个面向高端生活方式消费者的电商品牌,以「**高奢极简 · 东方哲学**」为品牌调性,围绕东方美学、现代极简、生活仪式感三个维度,为都市精英人群提供高品质的家居、生活、艺术商品。

我们不追逐快时尚的喧嚣,不堆砌促销弹窗,也不贩卖焦虑。屿音相信:
- **慢下来,才能看到生活的质感**
- **留白,本身就是一种奢华**
- **东方美学的当代演绎,不是复古,而是再生**

> *山峦不语,自有回响;器物无言,自成生活。*

## ✨ 品牌设计语言

### 🎨 五色克制 · 墨韵东方

| 色彩 | 名称 | HEX | 运用场景 |
|------|------|-----|----------|
| ▮ | **墨黑** (Ink Black) | `#1A1918` | 主文字、品牌主色、按钮、强调 |
| ▮ | **月白** (Moon White) | `#FAFAF9` | 主背景、卡片、留白 |
| ▮ | **金砂** (Gold Sand) | `#C9A962` | 辅助强调、点缀、价格 |
| ▮ | **竹青** (Bamboo Green) | `#4A7C59` | 成功状态、自然元素 |
| ▮ | **楮色** (Paper Brown) | `#8B7355` | 次要文字、辅助装饰 |

色彩不超过 5 种,克制中见气度,极简中显奢华。

### ✍ 字体 · 衬线为骨,无衬线为肉

- **Noto Serif SC** — 标题、品牌字、品牌叙事(传递东方书卷气)
- **Noto Sans SC** — 正文、辅助说明(保证可读性)
- 全部自托管,经字符子集化优化,体积从 11MB 降至 1MB

### ⬜ 留白 · 奢侈的呼吸感

- 8pt 网格系统贯穿全站
- 模块之间大量留白,让商品成为绝对焦点
- 去除一切冗余装饰,信息层级靠字体大小、字重、留白而非颜色

## 🛍 业务功能

屿音 Web 端整合六大业务模块,提供**一站式无缝购物体验**:

| 模块 | 功能 | 设计要点 |
|------|------|----------|
| 🏠 **首页** | 品牌叙事 + 精选系列 + 内容栏目 | 三段式滚动叙事,内容化电商 |
| 🛒 **商品浏览** | 分类树、搜索、筛选、收藏 | 极简商品卡片,大图为主,文字克制 |
| 🛍 **购物车** | 加购、改数、删品、凑单 | 迷你购物车 + 完整购物车,操作即时反馈 |
| 💳 **支付结算** | 多种支付方式、地址管理、发票 | 透明价格,无隐藏费用,3 步内完成 |
| 📦 **订单中心** | 订单列表、详情、物流跟踪、退换货 | 状态可视化,操作按钮精简,信息流清晰 |
| 👤 **会员中心** | 资料管理、地址簿、收藏、足迹、积分 | 极简双栏布局,信息分层展示 |
| 🎁 **营销活动** | 限时活动、优惠券、积分商城 | 克制营销,简化参与流程,规则轻量化展示 |

### 经营理念

- **不做价格战** — 我们相信品质的价值,不做喧嚣的促销
- **不堆砌推荐** — 首页只展示精选系列,商品卡片只用 1 张主图
- **不打扰用户** — 没有强制弹窗,没有消息轰炸,没有营销焦虑
- **内容即商品** — 品牌故事、生活方式、艺术栏目,与商品并重

## 📐 三端自适应

屿音 Web 端同时适配**桌面端、移动端 H5、微信小程序**,一套设计系统,三端无缝体验。

| 客户端 | 断点 | 特点 |
|--------|------|------|
| 🖥 **桌面端 (Desktop)** | 1440px+ | 完整顶部导航,左右双栏布局,商品大图居中 |
| 📱 **移动端 H5 (Mobile)** | 360px | 单列流式,底部 Tab 栏,触摸操作 ≥ 44px |
| 💬 **微信小程序** | 750rpx | 原生 TabBar + 自定义组件,主包 < 2MB |

> 小程序版仓库: `Yuyin-E-commerce-platform-fe/front-end/yuyin_miniprogram/`

## 🛠 技术栈

| 分类 | 技术 |
|------|------|
| 框架 | React 18 + TypeScript 5 |
| 构建 | Vite 5(manualChunks 智能分包) |
| 样式 | Tailwind CSS 3 + CSS Variables 主题令牌 |
| 路由 | React Router DOM 6 |
| 状态 | Zustand(全局) + TanStack React Query(服务端) |
| 网络 | Axios + 统一请求拦截器 |
| 三端 | DesktopLayout / MobileLayout / MiniProgramLayout |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器(默认 http://localhost:3000)
npm run dev

# 生产构建
npm run build

# 预览构建产物
npm run preview
```

## 🎨 设计规范

参考 [`./CLAUDE.md`](./CLAUDE.md),遵循:

- **品牌调性优先**:极简克制、留白为王、东方哲学
- **移动优先**:360px 断点,所有按钮 ≥ 44px 触控区
- **原子化组件**:单一职责、低耦合高内聚
- **性能优先**:路由级懒加载、字体子集化、关键资源 preload
- **可访问性**:WCAG 2.1 颜色对比度、语义化 HTML、键盘导航
